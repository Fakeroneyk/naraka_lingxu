## 阶段6：战斗单元测试

### 瞄准（训练场）
```bash
python -c "
import sys,time; sys.path.insert(0,'.')
from core.config import load_config
from capture.screen_capture import ScreenCapture
from input.mouse import Mouse
from strategy.aim_controller import AimController
from detector.enemy_detector import EnemyDetector
c=load_config(); cap=ScreenCapture(c); m=Mouse(); aim=AimController(c,m); det=EnemyDetector(c)
print('5秒后瞄准...'); time.sleep(5)
for i in range(30):
    f=cap.grab(); e=det.detect(f)
    if e: print(f'帧{i}: 目标({e[0].center_x},{e[0].center_y}) 瞄准={aim.aim_at(e[0])}')
    time.sleep(0.1)
"
```

### 问题排查

| 问题 | 排查 |
|------|------|
| 模板匹不到 | 重新截更清晰的模板 / 调低ui_confidence |
| 瞄准抖动 | 调低sensitivity_x/y |
| 传送门不识别 | 用HSV调试器校准颜色范围 |
| 按键无效 | 检查keybinds配置是否与游戏一致 |
| 坐标偏移 | 确认游戏窗口模式（无边框窗口最佳） |
